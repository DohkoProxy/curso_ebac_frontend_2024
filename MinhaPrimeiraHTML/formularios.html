O que aprendi?

Nesta etapa, apredio o uso de formularios (form). Dentro de form, aprendi a inserçao de caixa para formularios, sendo o primeiro de texto (text), onde deixei direcionado para um texto de nome com o placeholder ¨seu nome", que é a mensagem que vai aparecer na caixa de formulário. Depois aprendi o comando de email (input type=email), para esse aprendi a colocar um texto de separaçao antes da caixa de texto, para indicar o que o usuario deve preencher (label> Seu e-mail /label) , e para direcionar o usuario para caixa de texo ao clicar na indicaçao do label, inserimos seu id em label, (label for=email), também foi necessário inserir o id em input (input id=email), o que deixou o label assim : label for="email" > Seu E-mail < /label e o input assim : input id=email type="email".
Depois, aprendi o type de telefone (type=tel), que em computadores nao aparente ser tao diferente do primeiro campo de texto (text), entretanto, em dispositivos móveis, assim que voce selecionar esta caixa, o teclado que vai abrir é o numérico, facilitando para o usuário. E também aprendi a criar uma caixa numérica, o type=number. Que é uma caixa exclusiva para numeros, além de definir o aspecto máximo (max) o minimo (min) que o usuario pode selecionar e junto de quantos em quantos numero a contagem deve pular, por exemplo step=2 que vai contar de 2 em 2.
Em seguida aprendi a caixa para seleçao de arquivos (type=file) e seleçao de cor (type=color). Também aprendi a caixa de confirmaçao/marcaçao type=checkbox.
Também aprendi a fazer uma selecao, com diversas escolhas, com o comando type=radio além de aprender o name=pergunta em questao para poder dizer ao html que todas as alternarivas estao na mesma pergunta, impedindo assim de que o usuario possa selecionar todas as caixas. E em seguida aprendi o select que abre uma caixa com as opcoes que a pessoa pode selecionar com o comando options, permitindo assim, uma unica escolha do usuario, mas se for preferivel que possa escolher varias podemos usar o select multiple.
E por ultimo, aprendi a acrescentar botoes, com o comando button, que pode ser atribuido uma utilidade com o type selecionado.
